Recent Open Source Projects I've been focusing on Javascript and Visualization recently. Visit Github for more.

Encom Boardroom

An HTML5 recreation of the Boardroom Scene in Disney's Tron: Legacy. It currently displays realtime data from Github and Wikipedia to make it a bit more fun. View it in action at robscanlon.com/encom-boardroom.

  • Javascript
  • WebGL
  • Canvas
  • Server Sent Events
  • Node.js
  • JQuery

Encom Globe

A WebGL-based clone of the globe from the boardroom scene in Tron: Legacy. It was built for use by the Encom Boardroom project, but it is designed to be reused by anyone looking for an alternative to the popular WebGL Globe.

  • Javascript
  • WebGL
  • Three.js

Hexasphere.js

A javascript library with no dependencies that generates the geometry of a sphere covered in hexagonal tiles. It is used to create the Encom Globe, but could be used in a tile-based game or other "useful" applications.

  • Javascript

Streamed

A casual streaming data visualization platform for people to share meaningful data streams and creative visualizations that uses an IRC server as a message bus. It currently hosts the popular Github Wargames visualization.

  • Javascript
  • Node.js
  • Web Sockets
  • D3.js

HabitRPG-TXT

A text-based interface to HabitRPG that allows the service to be used in a console environment.

  • Javascript
  • Node.js
